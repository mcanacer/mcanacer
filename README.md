## What I've Built

### Generative Models
| Paper | Framework | Notes |
|---|---|---|
| [Stable Diffusion (LDM)](https://arxiv.org/abs/2112.10752) | JAX/Flax | Full from scratch including VAE + UNet + CLIP conditioning |

| [VQ-VAE](https://arxiv.org/abs/1711.00937) | JAX/Flax | Discrete codebook, full pipeline |
| [MaskGit](https://arxiv.org/abs/2202.04200) | JAX/Flax | Masked token prediction for image generation |
| [VAE](https://arxiv.org/abs/1312.6114) + [DC-GAN](https://arxiv.org/abs/1511.06434) | PyTorch | Foundational generative models |
